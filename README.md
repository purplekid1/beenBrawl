# BeenBrawl

Bean Brawl is a fast anime-inspired arena prototype where you control a bean fighter with robotic arms.

## Current gameplay loop
- Move, jump, and look around in first-person.
- Chain a 4-hit combo with arm animations.
- Hit the training dummy with raycast melee and impulse knockback.
- Spawn a particle hit effect when attacks connect.

## Controls
- `WASD` / arrow keys: move
- `Mouse`: look
- `Space`: jump
- `Left Mouse` or `F`: attack
- `Esc`: release mouse cursor
- `Left Mouse` (after releasing): capture cursor again
